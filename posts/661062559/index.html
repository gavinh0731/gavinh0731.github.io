  2. зӣӨж•ҙзӘҒз ҙ

еқҮз·ҡзңӢеқҮз·ҡеЈ“еҠӣж”Ҝж’җ

еқҮз·ҡзҡ„з®—жі•пјҢж„Ҹзҫ©

еқҮз·ҡпјҡж”¶зӣӨеғ№зҡ„е№іеқҮжҲҗжң¬

зҹӯгҖҒдёӯгҖҒй•·жңҹеқҮз·ҡжҮүз”Ёе…Ҙй–Җ

  1. жңҲз·ҡжҳҜзҹӯжңҹж“ҚдҪңзҡ„еӨҡз©әеҲҶз•Ңз·ҡ
  2. еӯЈз·ҡжҳҜдёӯй•·жңҹж“ҚдҪңзҡ„еӨҡз©әеҲҶз•Ңз·ҡ
  3. и¶ЁеӢўеӨҡй ӯеңЁжңҲз·ҡд№ӢдёҠпјҡжңҲз·ҡдёҠжҸҡ->еҒҡеӨҡдёҚеҒҡз©ә
  4. и¶ЁеӢўз©әй ӯеңЁжңҲз·ҡд№ӢдёӢпјҡжңҲз·ҡдёӢеҪҺ->еҒҡз©әдёҚеҒҡеӨҡ

еқҮз·ҡеӨҡй ӯжҺ’еҲ—

KзҘһдёҖжўқеқҮз·ҡж“ҚдҪңжі•

йҒёиӮЎпјҡз”ЁKDгҖҒMACDгҖҒKжЈ’гҖҒеқҮз·ҡ
з•¶йҒёдёӯиӮЎзҘЁйҖІе ҙд№ӢеҫҢпјҢи«ӢдҪ и¶Ҡз°Ўе–®и¶ҠеҘҪ

  1. йҖІе ҙпјҡж”¶зӣӨеүҚ(дёҖй»һд»ҘеҫҢ)зўәиӘҚиӮЎеғ№з«ҷдёҠеқҮз·ҡпјҲзҙ…KйҖІе ҙпјү
  2. еҒңжҗҚпјҡйҖІе ҙз•¶ж—ҘKз·ҡиӮЎеғ№зҡ„жңҖдҪҺй»һ(зҙ…Kзҡ„жңҖдҪҺй»һдёҚиҰҒи·Ңз ҙ)
  3. зәҢжҠұпјҡж”¶зӣӨеүҚжӘўиҰ–иӮЎеғ№пјҢжІ’жңүи·Ңз ҙеқҮз·ҡжҷӮзәҢжҠұ
  4. еҮәе ҙпјҡж”¶зӣӨеүҚзўәиӘҚиӮЎеғ№пјҢи·Ңз ҙеқҮз·ҡжҷӮеҮәе ҙ

е–®дёҖеқҮз·ҡжі•дҪҝз”Ёжўқд»¶

  1. з”Ёж–јеӨҡз©әи¶ЁеӢўжҳҺзўәжҷӮпјҢдёҚиғҪеңЁзӣӨж•ҙеҚҖй–“дҪҝз”Ё
  2. еҒҡеӨҡз¶ӯжҢҒеӨҡй ӯи¶ЁеӢўпјҢ20MAеқҮз·ҡдёҠжҸҡ
  3. еҒҡз©әз¶ӯжҢҒз©әй ӯи¶ЁеӢўпјҢ20MAеқҮз·ҡдёӢеҪҺ
  4. з„Ўи«–йҖІеҮәпјҢеқҮд»Ҙж”¶зӣӨеғ№зўәиӘҚ
  5. ж—Ҙз·ҡгҖҒйҖұз·ҡзҡҶеҸҜж“ҚдҪң

з¬¬5йӣҶпјҒпҪңKжЈ’зө„еҗҲж•ҷеӯё

KжЈ’жҳҜжҠҖиЎ“еҲҶжһҗд№ӢжҜҚпјҢе°ұжҳҜжҜҸеӨ©еӨҡиҲҮз©әдҪңжҲ°зҡ„зөҗжһңпјҢ

жҠ“дҪҸKжЈ’з·ҡзҙўпјҢзІҫжә–зҚІеҲ©

  1. KжЈ’й•·зӣёзңӢеҮәз•¶ж—ҘеӨҡгҖҒз©әеј·ејұ
  2. жҺҢжҸЎеӨҡгҖҒз©әзҹӯжңҹзҡ„иҪүжҠҳдҪҚзҪ®
  3. й ҳе…Ҳеҹәжң¬йқўгҖҒж¶ҲжҒҜйқўгҖҒжҢҮжЁҷжңҖеҝ«еҸҚжҮүеёӮе ҙи®ҠеҢ–пјҢеҸҜз«ӢеҚіж“¬е®ҡдәӨжҳ“зӯ–з•Ҙ
  4. зІҫйҖҡKжЈ’е®№жҳ“иІ·еңЁзӣёе°ҚдҪҺй»һгҖҒиіЈеңЁзӣёе°Қй«ҳй»һ
    е“ӘиЈЎжҳҜдҪҺй»һпјҹе“ӘиЈЎжҳҜй«ҳй»һпјҹ

KжЈ’еӣӣеӨ§е…ғзҙ 

иІ·иӮЎзҘЁпјҢж”¶зӣӨеүҚиІ·

  1. й–ӢзӣӨеғ№пјҡ
    • еҸҚжҮүеҲ©еӨҡжҲ–еҲ©з©әж¶ҲжҒҜйқў
    • зңӢеҮәдё»еҠӣеӨ§жҲ¶зҡ„дјҒең–пјҲиӮЎеғ№жҳҜдё»еҠӣжұәе®ҡзҡ„пјү
    • з•¶ж—Ҙи®ҠзӣӨй—ңйҚөпјҡй–Ӣй«ҳеҢ–и§Јз–‘ж…®гҖҒй–ӢдҪҺе°Ҹеҝғи®ҠзӣӨ
  2. жңҖй«ҳеғ№пјҡ
    • еӨҡж–№еҠӣйҮҸиғҪдёҠж”»зҡ„жңҖй«ҳеғ№дҪҚгҖҒжё¬и©ҰеЈ“еҠӣзҡ„дҪҚзҪ®
    • з©әй ӯжҠөж“ӢйҳІе®Ҳзҡ„еғ№дҪҚ
  3. жңҖдҪҺеғ№пјҡ
    • з©әж–№еҠӣйҮҸиғҪеҲ°йҒ”зҡ„жңҖдҪҺеғ№дҪҚгҖҒжё¬и©Ұж”Ҝж’җзҡ„дҪҚзҪ®
    • еӨҡй ӯжҠөж“ӢйҳІе®Ҳзҡ„еғ№дҪҚ
  4. ж”¶зӣӨеғ№пјҡ
    • еҸҜзңӢеҮәз•¶ж—ҘеӨҡз©әзҡ„еј·ејұпјҢй җжңҹжҳҺеӨ©й–Ӣй«ҳжҲ–й–ӢдҪҺзҡ„ж„Ҹең–
    • зӘҒз ҙжҳЁж—Ҙй«ҳй»һеүҮиҪүеј·пјҢзӘҒз ҙжҳЁж—ҘдҪҺй»һеүҮиҪүејұ
